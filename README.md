# CB audio processor / filter with DSP and ESP32
Second part of the series is about the CB audio processor / filter.

The speech processor repository has the links to the video and the instructions cover this project so make sure to read them. Same disclaimer applies.

A list of the audio processor / filter features: 

Low pass / high pass or bandpass filter.

Notch filters.

Peaking filter.

Dynamic noise filter.

Dynamic automatic notch filter.

Noise gate.

Compressor with adjustable threshold, ratio and post gain.

12 band equalizer.

Oscilloscope.

Spectrum analyzer.

Morse decoder.

10 presets.

Headphone amplifier.
